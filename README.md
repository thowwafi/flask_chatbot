# Flask Chatbot

## Installation

1. Clone this repository:
    ```bash
    git clone
    ```
2. Create Python Virtual Environment:
    ```bash
    python3 -m venv venv
    ```
3. Activate Virtual Environment:
    ```bash
    source venv/bin/activate
    ```
4. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
5. Run the app:
    ```bash
    flask --app app run
    ```
6. Open the app in your browser: http://
